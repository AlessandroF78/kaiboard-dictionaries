# KaiBoard Autocorrect Language Packs

Offline autocorrect dictionaries for KaiBoard multilingual keyboard.

## Supported Languages

| Language | Code | Dictionary Size |
|----------|------|----------------|
| Italian | it_IT | ~1.3 MB |
| German | de_DE | ~1.1 MB |
| Greek | el_GR | ~9.9 MB |
| English (US) | en_US | ~542 KB |
| Spanish | es_ES | ~864 KB |
| French | fr_FR | ~1.4 MB |
| Georgian | ka_GE | ~3.9 MB |
| Dutch | nl_NL | ~1.9 MB |
| Polish | pl_PL | ~4.7 MB |
| Portuguese | pt_PT | ~1.5 MB |
| Turkish | tr_TR | ~37.5 MB |
| Russian | ru_RU | ~3.5 MB |
| Ukrainian | uk_UA | ~8.6 MB |

## Dictionary Format

All dictionaries use the **Hunspell** format:
- .dic file: word list
- .aff file: affix rules

## Download

Download individual language packs from the [Releases](https://github.com/AlessandroF78/kaiboard-packs/releases) page.

## Usage in KaiBoard

KaiBoard automatically downloads the required language pack when you switch keyboard layout.
